## 标准
“很多时候我们都以当下自认的标准来判断一些事情，可是这个标准有可能是别人不承认的，有可能是自己之后或之前都不承认的，有可能根本就不是一个可以表述的标准。这时我们怎么去判断一件事的性质呢？我们又以什么样的出发点去和别人交流呢？”          
via @科学家种太阳        

“标准从来都不是标准的东西，定义的越多，离标准就越远。标准一定是简单的。只有简单的东西才具有穿透力”。           
via 簋谣(vczero)
